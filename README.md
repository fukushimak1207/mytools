# マイツール置き場（mytools）

自作の小さな Web ツールを1画面に集約する個人ハブ。単一の自己完結 HTML（外部リソースなし）で、GitHub Pages に置くだけで動きます。

## 内蔵ツール
- 🧠 **マインドマップ** — 枝分かれで考えを整理。Markdown / JSON 書き出し
- ✅ **チェックリスト** — 何度でも使える確認リスト（進捗バー・全部リセット）
- 🔍 **用語・略語メモ** — 自分だけの略語辞典（検索で絞り込み）
- 📝 **メモ帳** — 自動保存のかんたんメモ
- 🏷 ねだん帳 / 💰 家計簿 / 📋 検査当番表 へのカード（リンク・準備中）

## 使い方
- 公開ページ: https://fukushimak1207.github.io/mytools/
- ローカルで開く場合は `index.html` をブラウザで開くだけ

## メモ
- データは各ブラウザの localStorage に保存されます（PC 間では同期されません）。大事な内容は各ツールのコピー / 書き出し機能で持ち出してください。
- ライト / ダークテーマ対応、スマホ対応。
